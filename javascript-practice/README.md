# JavaScript Practice

A lightweight JavaScript practice project with a simple utility module and built-in tests.

## Quick Start

1. Open the `javascript-practice` folder in your editor.
2. Run `npm install` if you add third-party packages later.
3. Run `npm start` to execute the sample app.
4. Run `npm test` to execute the built-in test file.

## Project Structure

- `src/index.js` — sample entry point.
- `src/calculator.js` — reusable utility functions.
- `tests/test.js` — basic test harness using Node.js built-in `assert`.

## Practice Ideas

- Add new functions to `src/calculator.js`.
- Write new test cases in `tests/test.js`.
- Build a small CLI or browser-based app.
